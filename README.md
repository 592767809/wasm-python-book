### 第2章-二进制格式
完成了模块解码器和dump程序。
传入参数：
> -d "wasm-python-book\js\ch01_hw.wasm"

![](images/ch02/ch02.png)
1. 采用unittest进行单元测试
2. 由于使用小端格式读取数值，python可采用byteorder的little入参
```python
int.from_bytes(self.data[:8], byteorder='little')
```
3. 如果该结构体是数组，由于Python无法表示结构数组，故类初始化的时候初始一个数组。
